```

= += -= *= /=
%= **= >>= <<=
&&= ||= &= |= ^=

Comparison:
== != <= >= <=>
=== =~ !~

Range:
.. ...

Interpolation:
#{ #$ #@

Regex Quantifiers:
*? +? ?? }?
*+ ++ ?+

Regex Groups:
(?= (?! (?<= (?<!
(?: (?> (?# (?<
(?( (?~ (?-

Misc.:
[^ -> =>
```
